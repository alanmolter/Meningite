1. CORREÇÃO DO ERRO DE CORRELAÇÃO

1.1 Problema Identificado:
- Erro: "ValueError: All arguments should have the same length"
- Localização: Gráfico de correlação na aba "Faixa Etária"
- Causa: Incompatibilidade de tamanhos entre arrays de cobertura e incidência

1.2 Solução Implementada:
- Criação de merge adequado entre dados de cobertura e incidência
- Uso de DataFrame unificado para o gráfico de correlação
- Correção da sintaxe do plotly.express.scatter

1.3 Código Corrigido:
```python
# Merge dados de cobertura com dados de incidência para correlação
dados_correlacao = cobertura_clean.merge(
    casos_com_pop[['Regiao', 'Incidencia_por_100k']], 
    on='Regiao', 
    how='inner'
)

# Gráfico de correlação
fig_correlacao = px.scatter(
    dados_correlacao,
    x='Total_Pediatrico',
    y='Incidencia_por_100k',
    text='Regiao',
    title="Correlação: Cobertura Vacinal vs Incidência de Casos (2024)",
    labels={'Total_Pediatrico': 'Cobertura Vacinal Pediátrica', 'Incidencia_por_100k': 'Incidência por 100.000 habitantes'}
)
```
